JVM Tools

Essential Resources for Java Virtual Machine Development

Performance Monitoring and Profiling

  • JVM TI (JVM Tool Interface) - A native programming interface for inspecting and controlling the state of a running Java application.
  • async-profiler - A low overhead sampling profiler for Java featuring flame graphs.
  • YourKit Java Profiler - A feature-rich Java profiler for memory and CPU analysis.

Debugging and Diagnostics

  • jconsole - A monitoring tool for Java applications, included in the JDK.
  • JDK Mission Control - A powerful profiling and diagnostics tools suite for the JVM.
  • Vector - An on-host performance monitoring framework which exposes hand-picked high-resolution metrics to every engineer's browser.

Memory Management

  • Eclipse Memory Analyzer (MAT) - A fast and feature-rich Java heap analyzer.
  • Java Agent for Memory Measurements (JAMM) - A Java agent for measuring actual object memory use including JVM overhead.

Bytecode Manipulation

  • ASM - A Java bytecode manipulation and analysis framework.
  • cglib - A high-level API to generate and transform Java bytecode.

Build Tools and Dependency Management

  • Apache Maven - A popular build automation and project management tool.
  • Gradle - A flexible and powerful build automation tool.
